# Changelog — GEMS Language

All notable changes to the GEMS Language specification are documented here.

## [1.0.0] — 2025-01-26

### Added
- Initial public release of the GEMS Language specification.
- Library format: `library`, `port-types`, `models`, `ports`, `parameters`, `variables`, `constraints`, `binding-constraints`, `objective-contributions`.
- Study format: `system`, `components`, `connections`, `parameters`.
- Data-series support via time-dependent CSV parameters.
- Scenario builder support.
- Mathematical expression syntax for constraints and objectives.
