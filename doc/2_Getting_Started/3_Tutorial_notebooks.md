# Tutorial Notebooks

This section provides interactive Jupyter notebooks tutorials containing concrete of GEMS applications alongside results visualization.

These Jupyter notebooks provide basic examples showing how you can use GEMS concepts in a jupyter notebook on your local machine.

## Tutorial 1: Unit Commitment Example

This comprehensive tutorial shows how to build and solve a Unit Commitment problem using [GemsPy ](https://gemspy.readthedocs.io/en/latest/).

**Access the notebook:** [1_Tutorial_Unit_Commitment.ipynb](https://github.com/AntaresSimulatorTeam/GEMS/tree/main/resources/Documentation_Examples/Tutorial_1_Unit_Commitment)
